나만의 메모장 웹 사이트 (MyPersonalMemo) 기획안

[ 핵심 기능 ]
-> 로그인 / 로그아웃 기능을 통한 계정 별 투두 리스트 관리
-> 메모 제목, 세부 내용을 통해 업로드가 가능하도록 함. (작성 일자 저장)
-> 중요한 메모의 경우 [중요] 체크를 통해 우선적으로 띄워지도록 함.
-> 카테고리를 신설하고, 각 카테고리 별로 메모를 추가할 수 있도록 함.

[ 디자인 기능 ]

- 나도 구현해보자, 좌측에서 우측으로 나오는 사이드 메뉴
- 메모를 드래그 앤 드롭하여 다른 카테고리로 이전 가능하게끔 하는 애니메이션
- 카테고리가 많아질 경우, 이를 Carousel 로 구현하여 슬라이드로 보여지게 함.
- 메인 페이지 소개 글에 텍스트가 위로 올라오는 애니메이션도 구현해보기.
- 로그인 페이지로 넘어가는 것이 아닌, 로그인 화면을 modal로 구현해보기.

[ 기술적 요소 ]

- JWT 토큰 발급을 통한 로그인 구현 (세션 X)
- react-router-dom 을 활용한 동적 라우팅
- Kakao Rest API로 소셜 로그인 구현 시도
- MongoDB 와의 연동을 통한 유저 별 메모 Data 저장

[ 사용 기술 스택 ]

- React, Redux
- Express, MongoDB
